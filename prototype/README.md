## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | トップページ。検索導線、特集記事、新着情報、商工会ページ導線 |
| `businesses.html` | 会員企業一覧ページ。カード表示、カテゴリ/キーワード絞り込み |
| `business-detail.html` | 10万円プラン企業用の詳細ページ |
| `styles.css` | レイアウト・デザイン |
| `data.js` | サンプル企業・特集記事・インタビューのデータ |
| `app.js` | 記事表示、インタビュー表示、絞り込み、詳細表示、検索の動き |

## UI方向性

参考サイト: https://kumamoto-tsukigi.com/

トップページは、地域メディア型のUIを目指す。大きなコピー、写真グリッド、英字ラベル、NEWS、特集記事、経営者インタビューを重ね、行政的な情報掲示板ではなく「地域の人・店・仕事を読む」印象にする。

詳細方針は `../docs/ui-reference-direction.md` を参照。
